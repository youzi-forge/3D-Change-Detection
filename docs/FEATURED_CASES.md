# Featured cases (3RScan)

This page is a lightweight “portfolio view” for first-time readers.
It includes three representative report artifacts generated by this repo:

- two **reliable** pairs (clean overlays and localized change evidence)
- one **unreliable** pair (QC-gated failure mode: partial overlap / coverage mismatch)

If you want to reproduce the same artifacts locally, run:

```bash
python3 scripts/run_batch.py --datasets-root Datasets --pairs-json configs/pairs/featured.json --out-root outputs/featured --exclude-labels wall,floor,ceiling --resume
```

## Failure case (unreliable by QC gate)

- pair_id: `280d8ebb-6cc6-2788-9153-98959a2da801__4731976c-f9f7-2a1a-95cc-31c4d1751d0b`
- QC: overlap_gate_value = 0.246 (< overlap_min = 0.300), overlap_mean = 0.344, comparable_ref = 0.602, comparable_rescan = 0.994
- Interpretation: this is a classic partial-overlap case. A large fraction of the reference scan is not observed in the rescan (and vice versa),
  so naive differencing produces many “appeared/disappeared” objects. The pipeline intentionally marks this as **unreliable** and treats downstream outputs as diagnostics.
- Full report: `docs/featured_cases/280d8ebb-6cc6-2788-9153-98959a2da801__4731976c-f9f7-2a1a-95cc-31c4d1751d0b/report.html`
